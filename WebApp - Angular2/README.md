
ag-Grid-ng2-examples - systemjs & systemjs-builder
==================================================

Example of using ag-Grid with Angular2 and TypeScript

Building
========

Install Dependencies:

- `npm install gulp -g`
- `npm install`

JIT Build
=========

To build:

- `npm run tsc`

To Run:

- `npm run lite`

AOT Build
=========

To build:

- `npm run clean-build:aot`

To Run:

- `npm run lite:aot`
